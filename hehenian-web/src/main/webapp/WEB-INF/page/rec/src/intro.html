• 100%本息回购

    合和年在线通过要求资金雄厚的第三方机构和融资项目推荐机构对平台的融资项目进行逾期回购，让融资项目必须具有一定的信誉、实力和可控的预期，切实保证投资人的100%投资本息。

  • 网络技术保障个人信息安全

    一、支持安全套接层协议128和位加密技术——这种加密技术是互联网上保护数据安全的行业标准，让客户在进行会员管理、个人账户管理、充值等涉及敏感信息的操作时，信息被自动加密，然后才被安全地通过互联网发送出；

    二、合和年在线在任何时候竭力保证客户的个人信息不被人擅自或意外取得、处理或删除。采取各种实际措施保证个人信息不会被保管超过合理的期限,并设有严格的安全系统，以防止未经授权的任何人、包括本公司的职员获取客户信息。本公司将遵守所有关于可辨识个人信息保存的法规要求。

  • 当一个借款标的在合和年在线发布之前，需要经过3道审核。

    第一步：融资项目推荐机构调查 所有的合和年在线融资项目都要经过实地尽职调查，由当地的融资项目推荐机构执行。 调查报告的数据包括实地调查数据、人民银行征信系统数据、公安部居民身份系统数据、国防部安全信息系统数据、税务系统数据、海关系统数据、工商局系统数据、车辆管理系统数据、房屋管理系统数据等，并根据数据对融资项目进行风险定级。 第二步：融资项目推荐机构电话审核 当取得调查数据后，由融资项目推荐机构总部风险控制部门进行电话审核，电话审核内容包括借款人本人核实，借款人的软信息如品质、稳定性，借款人工作信息，借款人的社会关系等。 第三步：合和年在线风险评估 当推荐机构将融资项目推荐至合和年在线后，由合和年在线对项目核实信息的真实性与准确性并进行再次的风险评估与预期收益确定。当项目在收益和风险上都符合了优质理财的标准后，会被投放到平台供投资人选择。

  • 国内一流的贷后管理团队

    公司具有高素质、高学历、专业化的贷后管理团队，贷后管理团队包括金融、理财、咨询、财务、法律等方面的优秀专业人才，都具备了丰富的金融行业和小额信贷行业的工作经验，具备科学、严格、完善的风险控制体系。

  • 第三方资金托管

    一、投资人、借款人、合和年在线都在汇付天下开立独立控制的第三方托管账户以用于合和年在线平台上的资金流转，三方资金严格分离；

    二、投标时，经本人确认，投资人的资金直接由投资人的汇付天下托管账户转至借款人的汇付天下账户；还款时，经本人确认，还款资金由借款人的汇付天下托管账户直接转至投资人的汇付天下托管账户；
